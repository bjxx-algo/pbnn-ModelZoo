◆ Detect()

bool camera::YoloV8sDetector::Detect ( std::string  image,
DetectionResult &  det_result,
bool  draw_save_image = false 
)

对输入图像进行 YOLOv8 检测

该函数接收图像路径作为输入，完成图像读取、预处理、模型推理及后处理， 并将检测结果存储在 det_result 中。 可选地在图像上绘制检测框并保存结果。

参数
[in]image输入图像的文件路径
[out]det_result检测结果结构体，包含每个目标的类别、置信度和边界框坐标
[in]draw_save_image是否在图像上绘制检测结果并保存，默认为 false
返回
true 表示检测成功
false 表示检测失败（如图像读取失败或模型推理异常）
