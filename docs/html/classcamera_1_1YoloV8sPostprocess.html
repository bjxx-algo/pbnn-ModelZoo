YOLOv8 模型的后处理函数

该函数用于对模型推理输出结果进行后处理，包括解码检测框、筛选目标、绘制结果等操作。 可选地将结果绘制到原图上并保存。

参数
[out]out_data模型推理输出数据结构体，包含输出 tensor 数据及长度信息
[in,out]cv_image输入图像（BGR 格式），当 draw_save_image 为 true 时， 函数会在该图像上绘制检测框
[out]det_result检测结果结构体，包含每个目标的类别、置信度和边界框坐标
[in]draw_save_image是否在图像上绘制检测结果并保存，默认为 false
返回
true 表示后处理成功
false 表示后处理失败（如数据为空或解析错误）
